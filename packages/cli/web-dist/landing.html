Everything you need to cut AI costs

A complete toolkit spanning CLI, browser extension, web dashboard, and SDK.

🧠

Intent Detection

Classifies prompts into 6 intent types — debug, code-gen, refactor, explain, creative, general — with confidence scoring. Optimization adapts automatically per intent.

🎯

Token Budget Targeting

Set a target token count and auto-select the right compression level (1–4) to hit your budget without losing meaning.

✏️

Smart Prompt Rewriter

4-pass rewriter — verbose phrases, sentence compression, voice transform, question restructuring — gated by detected intent to avoid over-optimizing.

#

Token Counter

Exact token counts for OpenAI models via tiktoken, ~94% accurate for Claude. Know your token cost before you send.

💲

Cost Calculator

Real-time cost estimates with per-model pricing for 14 models — GPT-4o, o1, o3, Claude Opus/Sonnet/Haiku and more.

⚡

Context Monitor

Visual progress bar showing how much of your context window you've used, with color-coded warnings as you approach the limit.

💡

Strategy Advisor

Analyzes your project config, conversation history, model usage, and prompt patterns — then surfaces actionable token-saving recommendations.

🗂️

Cache Analysis

Clusters your prompts to identify prompt-prefix caching opportunities. Shows which clusters benefit most and provides a step-by-step setup guide.

📊

Web Dashboard

Browser-based dashboard with Insights, Analyze & Optimize, History, and Strategies tabs — plus Claude Code usage stats and cost tracking.

🌐

Chrome Extension

Floating widget on ChatGPT and Claude that shows live token count and cost estimate as you type — before you hit send.

⌨️

Interactive TUI

Run promptfuel with no arguments to launch a full terminal UI — analyze, optimize, and browse history without leaving the shell.

📦

Batch Processing

Analyze or optimize multiple prompts at once from a JSON file. Ideal for auditing prompt libraries or pre-processing datasets offline.

Use it programmatically

npm SDK for Node.js apps — analyze, optimize, and monitor tokens in your codebase.

sdk-example.ts 
import { optimize, detectIntent, calculateCost, monitorContext } from '@promptfuel/sdk';

// Optimize a prompt — intent detected automatically
const result = optimize(
  'I would like you to please explain how database indexing works in detail',
  { model: 'gpt-4o' }
);

console.log(result.intent);           // { type: "explain", confidence: 0.6 }
console.log(result.optimizedPrompt);  // "Explain how database indexing works in detail"
console.log(result.reductionPercent); // 42

// Check cost before sending
const cost = calculateCost(result.optimizedPrompt, 'gpt-4o');
console.log(cost.inputCost);          // "$0.000023"

Quick Start

Up and running in under a minute.

Install

$ pnpm install && pnpm build

Optimize a prompt

$ npx promptfuel optimize "I would like you to please explain how React hooks work in detail"

With a token budget

$ npx promptfuel optimize "Please help me debug this error step by step" --budget 10

Analyze project for savings

$ npx promptfuel strategies ./my-project --model claude-sonnet-4-6

Launch web dashboard

$ npx promptfuel dashboard

Interactive TUI

$ npx promptfuel
